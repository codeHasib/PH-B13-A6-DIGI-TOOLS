# 🧠 Digi-Tools

## 📌 Description

Digi-Tools is a modern web application where users can explore and purchase subscriptions for various AI tools. It provides a smooth and interactive experience for selecting tools, managing a cart, and viewing the total subscription cost in real-time. The project focuses on clean UI design and user-friendly functionality.

---

## 🚀 Technologies Used

- ⚛️ React
- 🎨 Tailwind CSS
- 🌼 DaisyUI

---

## ✨ Features

- 🛒 Users can add AI tool subscriptions to their cart (dummy purchase system)
- ❌ Ability to remove selected subscriptions from the cart
- 💰 Real-time total price calculation and display

---

## 📦 Installation & Setup

```bash
git clone https://github.com/codeHasib/digi-tools.git
cd digi-tools
npm install
npm run dev
```
